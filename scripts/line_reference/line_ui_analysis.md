# LINE 電腦版 UI 框架分析

## 共通區域（三張圖都有）

### 右上角 — 群組名稱/聊天標題（黃框/紅框/橘框）
- 顯示當前開啟的對話名稱：「好朋友的群組 (66)」
- 括號內數字 = 群組人數
- 旁邊有靜音、通話、視訊、筆記、選單等按鈕

### 右側 — 對話區/聊天內容（綠框/藍框）
- 顯示所有對話訊息
- 包含：日期分隔線（4月10日、4月17日）
- 系統訊息（誰加入群組）
- 對方訊息（頭像+名字+內容+時間，靠左）
- 自己訊息（靠右）

### 右下角 — 輸入框（藍框/紅框）
- 「輸入訊息」文字提示
- 下方有附件、Keep筆記、截圖、表情符號按鈕

### 最左側 — 功能選單列（固定不變）
- 好友（人形圖示）
- 聊天（對話框圖示）
- 社群（多人圖示）
- 動態消息
- 通話
- Keep
- LINE官方
- 更多

## Line1 — 好友分類列表（左側面板）

這張是點了「好友」圖示後的畫面，左側面板依分類顯示：

| 顏色框 | 區域 | 內容 |
|--------|------|------|
| 綠框 | 搜尋欄 | 「以姓名搜尋」— 搜尋好友用 |
| 紅框 | 置頂好友/最近互動 | 「佳瑩」— 顯示頭像+名字+狀態消息 |
| 紫框 | 社群區塊 | 「社群 1」底下列出「領導力學苑 (三)」|
| 粉框 | 群組區塊 | 「群組 1」底下列出「好朋友的群組 (66)」|
| 棕框 | 好友區塊 | 「好友 2」底下列出「LINE」官方帳號 +「秋愷」|

**結構：好友頁把聯絡人分成三層 — 社群 / 群組 / 好友**

---

## Line2 — 聊天列表（左側面板）

這張是點了「聊天」圖示後的畫面，左側面板顯示最近聊天記錄：

| 顏色框 | 區域 | 內容 |
|--------|------|------|
| 綠框 | 搜尋欄 | 「搜尋聊天和訊息」— 搜尋對話記錄用 |
| 藍框 | 聊天列表 | 依時間排序的對話列表：Keep筆記、領導力學苑(三)(824則未讀)、好朋友的群組(66) |
| 紅框 | 聊天標題 | 「好朋友的群組 (66)」 |

**頂部有分頁標籤：全部 / 群組 / 社群 / 篩選**

**結構：聊天頁顯示所有最近對話，每個項目有頭像+名稱+最新訊息預覽+時間+未讀數**

---

## Line3 — 加好友頁面（左側面板）

這張是點了「加好友」圖示後的畫面：

| 顏色框 | 區域 | 內容 |
|--------|------|------|
| （上方無框）| 功能按鈕 | 搜尋好友、建立群組、建立社群 — 三個操作入口 |
| （中間無框）| 官方推薦 | 「官方推薦 2」— 歐c物理治療師、法律白話文運動 |
| 黑框 | 你可能認識的人 | 「您可能認識的人？1」— 泊瀾（透過電話號碼將您設為好友）|
| 藍框 | 對話區 | 同樣的聊天內容 |
| 紅框 | 輸入框 | 輸入訊息 |

**結構：加好友頁提供搜尋/建立群組/建立社群的入口，以及系統推薦的聯絡人**

## 整體 LINE 框架結構

```
+--+------------------+-------------------------------+
|  |                  |  聊天標題（群組名/好友名）      |
|  |    搜尋欄        +-------------------------------+
|功|                  |                               |
|能|                  |                               |
|選|    左側面板       |        對話區                  |
|單|  (依功能切換)     |     （聊天訊息內容）            |
|列|                  |                               |
|  |  - 好友分類列表   |                               |
|  |  - 聊天記錄列表   |                               |
|  |  - 加好友頁面     +-------------------------------+
|  |                  |  輸入框 + 附件/表情按鈕         |
+--+------------------+-------------------------------+
```

## 關鍵 UI 元素座標邏輯

1. **左側面板**寬度固定，內容依點擊的功能圖示切換
2. **分隔線**在左側面板和對話區之間
3. **對話區**佔右側大部分空間
4. **輸入框**固定在右下角
5. **聊天標題**固定在右上角
6. **功能選單列**固定在最左邊（約40px寬）
