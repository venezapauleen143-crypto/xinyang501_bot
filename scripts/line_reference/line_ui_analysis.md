# LINE 電腦版 UI 框架分析

## 共通區域（三張圖都有）

### 右上角 — 群組名稱/聊天標題（黃框/紅框/橘框）
- 顯示當前開啟的對話名稱：「好朋友的群組 (66)」
- 括號內數字 = 群組人數
- 旁邊有靜音、通話、視訊、筆記、選單等按鈕

### 右側 — 對話區/聊天內容（綠框/藍框）
- 顯示所有對話訊息
- 包含：日期分隔線（4月10日、4月17日）
- 系統訊息（誰加入群組）
- 對方訊息（頭像+名字+內容+時間，靠左）
- 自己訊息（靠右）

### 右下角 — 輸入框（藍框/紅框）
- 「輸入訊息」文字提示
- 下方有附件、Keep筆記、截圖、表情符號按鈕

### 最左側 — 功能選單列（固定不變）
- 好友（人形圖示）
- 聊天（對話框圖示）
- 社群（多人圖示）
- 動態消息
- 通話
- Keep
- LINE官方
- 更多

**結構：好友頁把聯絡人分成三層 — 社群 / 群組 / 好友**

## 整體 LINE 框架結構

```
+--+------------------+-------------------------------+
|  |                  |  聊天標題（群組名/好友名）      |
|  |    搜尋欄        +-------------------------------+
|功|                  |                               |
|能|                  |                               |
|選|    左側面板       |        對話區                  |
|單|  (依功能切換)     |     （聊天訊息內容）            |
|列|                  |                               |
|  |  - 好友分類列表   |                               |
|  |  - 聊天記錄列表   |                               |
|  |  - 加好友頁面     +-------------------------------+
|  |                  |  輸入框 + 附件/表情按鈕         |
+--+------------------+-------------------------------+
```

## 關鍵 UI 元素座標邏輯

1. **左側面板**寬度固定，內容依點擊的功能圖示切換
2. **分隔線**在左側面板和對話區之間
3. **對話區**佔右側大部分空間
4. **輸入框**固定在右下角
5. **聊天標題**固定在右上角
6. **功能選單列**固定在最左邊（約40px寬）

## Line7 — 搜尋好友結果頁面

搜尋好友後的畫面，搜尋結果出現在搜尋欄下方：

| 元素 | 位置 |
|------|------|
| 搜尋欄（已輸入好友名） | 面板最上方 |
| 「好友 1」分類標題 | 搜尋欄下方 |
| 好友項目（黑色框）| 分類標題下方，包含頭像+名字+狀態 |

**結構：搜尋後要點擊黑色框的好友項目才能進入對話**

## 分享好友資訊完整流程

```
1. 好友頁 → 搜尋好友（如溫妮）
2. 右鍵點擊溫妮
3. OCR 找「分享好友資訊」→ 點擊（line8）
4. 獨立面板「選擇傳送對象」出現在 LINE 左邊外側（line9 黃框）
5. 用 win32gui 找到面板視窗座標（約 352x552）
6. 點搜尋欄 → 輸入目標好友名（line10 黑框，x=0.500, y=0.195）
7. 點圓圈勾選好友（line14 紅點，x=0.926, y=0.341）
8. 重新 win32gui 抓面板尺寸（面板變高 552→600）
9. 點「分享」按鈕（line12 紅框，x=0.341, y=0.938）
```
